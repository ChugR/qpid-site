Detailed Description

template<class K, class T>
class proton::map< K, T >

Used to access standard AMQP property, annotation and filter maps attached to proton::message and proton::source.

Provides only basic get()/set() operations for convenience. For more complicated use (iteration, preserving order etc.) you should convert to a standard C++ map type such as std::map. See message_properties.cpp and AMQP and C++ types.
