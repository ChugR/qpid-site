Detailed Description

url is a proton URL of the form <scheme>://<username>:<password>@<host>:<port>/<path>. scheme can be amqp or amqps. host is a DNS name or IP address (v4 or v6) port can be a number or symbolic service name like amqp. path is normally used as a link source or target address, on a broker it typically it corresponds to a queue or topic name.

Examples:
broker.cpp, client.cpp, direct_recv.cpp, direct_send.cpp, helloworld.cpp, helloworld_blocking.cpp, helloworld_direct.cpp, server.cpp, server_direct.cpp, simple_recv.cpp, simple_send.cpp, and sync_client.cpp.
