Detailed Description

messaging_handler base class. Provides a simpler set of events than proton::proton_handler and automates some common tasks. Subclass and over-ride event handling member functions.

See Also
proton::messaging_event for meaning of events.
Examples:
broker.cpp, client.cpp, direct_recv.cpp, direct_send.cpp, helloworld.cpp, helloworld_direct.cpp, server.cpp, server_direct.cpp, simple_recv.cpp, and simple_send.cpp.
