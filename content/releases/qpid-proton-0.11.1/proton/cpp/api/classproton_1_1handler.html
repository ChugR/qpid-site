Detailed Description

Base class for event handlers.

A handler can have child handlers which are called in order, after the parent handler.

Note: handlers are not deleted automatically. They must not be deleted while they are still in use.

There are two simple strategies you can use:

  1. Destroy handlers only after the container that uses them is closed.
  2. Allocate handlers with new and call delete this in the appropriate on_*_closed or on_*_final event that indicates the handler is no longer needed.
