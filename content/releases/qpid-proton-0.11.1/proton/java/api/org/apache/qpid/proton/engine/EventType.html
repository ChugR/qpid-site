Interface EventType

  • All Known Implementing Classes:
    Event.Type

    public interface EventType
    Entry point for external libraries to add event types. Event types should be final static fields. EventType instances are compared by reference.

    Event types are best described by an enum that implements the EventType interface, see Event.Type.
