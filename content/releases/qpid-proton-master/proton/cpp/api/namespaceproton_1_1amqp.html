Detailed Description

AMQP typedefs for C++ types.

This namespace contains typedefs to associate AMQP scalar type names with the corresponding C++ types. These are provided as a convenience for those familiar with AMQP, you do not need to use them, you can use the C++ types directly.

The typedef names have a _type suffix to avoid ambiguity with C++ reserved and std library type names.
