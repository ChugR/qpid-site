Qpid Proton 0.7 Release Notes

Proton is a high-performance, lightweight messaging library. More about Qpid Proton.

For more information about this release, including download links and documentation, see the release overview.

New features and improvements

  • PROTON-287 - Fix Qpid Proton-C build with MinGW on Fedora
  • PROTON-343 - Add a pluggable Proton logging layer
  • PROTON-439 - Support for dynamic reply-to address in Messenger
  • PROTON-467 - Stop initializing encoder and decoder for each message sent/received
  • PROTON-476 - Support a user-context for SASL and SSL objects.
  • PROTON-496 - [Messenger] Messenger fails to send "end" on shutdown.
  • PROTON-498 - Set VALGRIND environment variable to be compatible with the QPID build
  • PROTON-502 - [proton-j] add UnsignedLong.valueOf(BigInteger)
  • PROTON-507 - [proton-c] Idle time out should advertise half the actual timeout period.
  • PROTON-515 - Port to OpenVMS
  • PROTON-518 - Add SASL hostname getter/setter
  • PROTON-526 - C API-doc is not well organized
  • PROTON-527 - events api for proton-j
  • PROTON-533 - Minor cleanup for 0.7 release
  • PROTON-536 - Make exported cmake files more idiomatic
  • PROTON-557 - Include local endpoint state changes in the engine Event API.
  • PROTON-565 - Modify the Messenger to use the Collector API.
