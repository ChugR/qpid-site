Qpid JCA

A Java EE Connector Architecture 1.5 compliant resource adapter that allows for JEE integration between EE applications and AMQP 0-10 message brokers.

  • Languages - Java
  • Platforms - JVM
  • AMQP versions - 0-10
  • Download - qpid-java-6.0.3.tar.gz [ASC, MD5, SHA1]
  • Source location - http://svn.apache.org/repos/asf/qpid/java/trunk/jca/

Features

  • JCA 1.5 compliant
  • Outbound and inbound connectivity
  • Connects to JBoss EAP, Geronimo, and Glassfish
  • Supports binding to Swing and other non-managed clients
