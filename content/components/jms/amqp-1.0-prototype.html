Earlier AMQP 1.0 JMS Client

Before the new Qpid JMS AMQP 1.0 client became available, the Qpid project offered an earlier AMQP 1.0 JMS client based on some initial prototyping work. Its details are preserved here to support existing users during transition to the newer client. If you are using this earlier client or just starting out, we encourage you to look at the newer Qpid JMS AMQP 1.0 client.

Download

The earlier AMQP 1.0 JMS client is released as part of Qpid 0.32.

  • PGP signature

Maven

The earlier AMQP 1.0 JMS client is also available via Maven. Use the following depencencies:

<dependency>
  <groupId>org.apache.qpid</groupId>
  <artifactId>qpid-amqp-1-0-client-jms</artifactId>
  <version>0.32</version>
</dependency>
<dependency>
  <groupId>org.apache.geronimo.specs</groupId>
  <artifactId>geronimo-jms_1.1_spec</artifactId>
  <version>1.0</version>
</dependency>
